# 📌 Project Overvieww

This project demonstrates an **AI-powered conversational chatbot** built using **Zoho SalesIQ's chatbot builder** and integrated with a **Shopify e-commerce website**.

The chatbot is designed to automate communication with users visiting the website and provide instant assistance through structured conversation workflows.

By using **block-based chatbot workflows**, the system can process user inputs and deliver responses automatically without human support.

This improves:

✔ Customer engagement  
✔ Response time  
✔ Support efficiency  

---

The main goals of this chatbot project are:

• Automate customer conversations
